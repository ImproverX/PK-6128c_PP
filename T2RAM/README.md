# Модуль ОЗУ-Т2 для ПК-6128ц++

Альтернативный модуль памяти с возможностью установки задержки процессора кратной двум тактам вместо стандартных четырёх.
Основное отличие схемы от стандартного модуля -- установка сдвиговых регистров 74HC597, что даёт возможность разнести такты обращения графической подсистемы к ОЗУ.

Установка данного модуля взамен стандартного возможна в любой версии плат ПК-6128ц++, но в схемах до 27.01.2025, из-за отсутствия в схеме ОЗУ-Т2 сигнала /CAS, не будут работать внешние квази-диски (на ВУ).

По тестам, выравнивание с кратностью равной двум тактам, даёт основное преимущество на командах:
- CALL, C*-Y/N
- DAD
- DCX, INX
- PUSH
- R*-Y/N
- SPHL
- RST*+RET

А также на недокументированных командах процессора 8085:
- DSUB
- LDHI, LDSI
- RDEL
- RSTV

На плате предусмотрен выключатель JP3, который переключает режимы работы ОЗУ. В текущей реализации прошивки в положении "включено" будет торможение процессора в 4 такта, в положении "выключено" -- 2 такта.
